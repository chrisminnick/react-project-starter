How to use our automated build system.

Building

  1. Open a command line and go to the project director.
  2. Type 'gulp karma' to start the karma server.
  3. Open at least one web browser to http://localhost:9876.
  4. Type 'gulp' to run the tests and build script.

Running

  1. Type 'gulp run' into the command line.
  2. Open a web browser to http://localhost:8080.
